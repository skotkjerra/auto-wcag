Rule: SC3-1-1-xml-lang

Late updated: Apr 11, 2018

Created by: Annika Nietzio

Accessibility Requirements

This conformance rule relates to:

  • Success Criterion 3.1.1 (Language of Page)
    • How to Meet 3.1.1 (Language of Page)
    • Understanding Success Criterion 3.1.1 (Language of Page)

Description

This test checks the value of the xml:lang attribute.

Background

  • H57: Using language attributes on the html element
  • eGovMon test for H57
  • BCP 47: Tags for the Identification of Languages

Assumptions

  • Tests have shown that xml:lang is ignored by screenreaders. (Both Jaws 15 with FF and IE and NVDA with FF go by lang attribute, xml:lang is ignored.) This test identifies pages that use only the xml:lang attribute.

Test procedure

Selector

Select all elements that matches the following CSS selector:

html[@xml:lang]

For each selected item, go through the following steps:

Step 1

Check that the html element contains also a lang attribute.

If no lang is specified, return step1-fail

Step 2

L1 = value of xml:lang

L2 = value of lang

Compare L1 and L2.

If the L1 and L2 differ, return step2-fail

Else, return step2-pass

Outcome

The resulting assertion is as follows,
